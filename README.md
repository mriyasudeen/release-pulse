# Release Pulse

Release Pulse is a lightweight release tracking service created to explore modern cloud-native development patterns.

The project allows engineering teams to record deployment outcomes, release confidence and operational notes through a simple API.

## Current Stack

- Node.js
- Express
- Local JSON storage

## Planned AWS Features

- AWS Lambda
- API Gateway
- DynamoDB
- Terraform
- GitHub Actions

## Goals

- Learn Infrastructure as Code
- Explore Serverless Architecture
- Build CI/CD Pipelines
- Gain hands-on AWS experience
